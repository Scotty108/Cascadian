**After:** 2 seconds - WIS badge, PnL, win rate, ranks all above fold

### Mobile Experience
**Before:** Endless scrolling on mobile
**After:** Critical info first, expansion optional

## Design Principles Applied

1. **F-Pattern Scanning**
   - Most important info in top-left
   - Metrics in horizontal row for quick scan
   - Secondary info flows down left side

2. **Progressive Enhancement**
   - Core experience: top metrics + primary chart
   - Enhanced: secondary charts for those who scroll
   - Advanced: OHLC, full tables for power users

3. **Information Scent**
   - "Show All X" buttons tell users what they'll get
   - Counts in accordion headers
   - Category PnL totals guide exploration

4. **Visual Weight**
   - Primary charts: full width, taller
   - Secondary charts: 50% width, shorter
   - Tertiary data: collapsed/truncated

5. **Cognitive Load Reduction**
   - Group related info (DNA badges together)
   - Collapse non-essential (OHLC)
   - Truncate long lists (top 5/10)
